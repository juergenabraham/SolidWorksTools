# SolidWorks 2024 SP05 — Tools & Macros

A collection of Python-based tools and macros for **SolidWorks 2024 SP05**.

**Platform:** Windows
**Language:** Python 3.10+

## Getting Started

**First-time setup:** navigate into the tool folder you want and run `setup.bat`.

**Staying up to date:** run `update.bat` from the root folder — pulls the latest
changes from GitHub and refreshes dependencies for all installed tools automatically.

```bat
update.bat
```

## Available Tools

| Tool | Version | Description |
|---|---|---|
| [SW2024-05-2STEP](SW2024-05-2STEP/README.md) | 0.5.0 | Convert SolidWorks files (`.sldprt`, `.sldasm`) to STEP format |
